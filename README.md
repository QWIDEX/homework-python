# 💸 Фінансовий трекер студента

Консольний чат-бот на Python для відстеження особистих витрат та бюджету студента.

---

## Концепція бота

### Як зберігаються дані

Усі дані (бюджет та список витрат) зберігаються у файлі **`finance_data.json`** у тій же папці, де запущена програма.

Структура файлу:
```json
{
  "budget": 5000.0,
  "expenses": [
    {
      "amount": 150.0,
      "category": "їжа",
      "date": "01.06.2025",
      "comment": "обід у їдальні"
    }
  ]
}
```

- При **запуску** програма зчитує дані з файлу (або створює порожню структуру).
- Після **кожної зміни** (додавання витрати, встановлення бюджету) дані автоматично записуються у файл.

---

### Як обробляються команди

Програма використовує **словник команд** (`dict`), де ключем є рядок команди, а значенням — відповідна функція. Головний цикл зчитує ввід користувача, приводить його до нижнього регістру та шукає збіг у словнику.

---

## Загальний алгоритм роботи

1. **Запуск програми** → завантаження даних із `finance_data.json`
2. **Вивід вітання** та підказки про команду «допомога»
3. **Головний цикл (while True):**
   - Зчитати ввід користувача
   - Якщо `вийти` → завершити програму
   - Якщо команда є у словнику → викликати відповідну функцію
   - Інакше → повідомити про невідому команду
4. **Після кожної зміни даних** → автоматичне збереження у файл
5. **Після кожного додавання витрати** → перевірка на перевищення бюджету

---

## Список команд

| Команда               | Опис                                      |
|-----------------------|-------------------------------------------|
| `допомога`            | Показати список команд                    |
| `встановити бюджет`   | Задати загальний бюджет                   |
| `додати витрату`      | Записати нову витрату                     |
| `показати витрати`    | Переглянути всі витрати                   |
| `фільтр дата`         | Витрати за конкретну дату                 |
| `фільтр період`       | Витрати за діапазон дат                   |
| `фільтр категорія`    | Витрати за категорією                     |
| `залишок`             | Скільки коштів залишилось                 |
| `звіт за категоріями` | Загальна сума витрат по категоріях        |
| `вийти`               | Завершити роботу програми                 |
